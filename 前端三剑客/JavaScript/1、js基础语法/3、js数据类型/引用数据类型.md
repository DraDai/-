### 引用数据类型

- **对象（Object）：** 表示复杂数据结构，可以包含多个键值对。

  ```javascript
  var person = {
      name: 'John',
      age: 25,
      isStudent: true
  };
  ```

- **数组（Array）：** 表示一组有序的值的集合。

  ```javascript
  var numbers = [1, 2, 3, 4, 5];
  ```

- **函数（Function）：** 表示可执行的代码块，可以被调用执行

  ```javascript
  function greet(name){
      console.log('Hello, ' + name + '!');
  }
  ```

- **日期（Date）：** 表示日期和时间的数据类型

  ```;
  var today = new Date();
  ```

- **正则表达式（RegExp）：** 表示用于匹配字符串的模式。

  ```javascript
  var pattern = /hello/;
  ```
