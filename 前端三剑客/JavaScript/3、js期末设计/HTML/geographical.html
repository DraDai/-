———— 地理环境 ————

位置境域

九江市位于江西省北部，长江中下游北岸，东临安徽省安庆市、铜陵市，南接江西省瑞昌市、庐山市，西连湖北省黄梅县、武穴市，北濒长江，与安徽省芜湖市隔江相望。九江市全境东西长约180公里，南北宽约120公里，总面积约14744平方公里。

九江市辖浔阳区、濂溪区、柴桑区、德安县、修水县、武宁县、彭泽县、星子县、都昌县、共10个区县。市政府驻浔阳区。

地貌地形

九江市地貌以丘陵为主，山地、平原、水面相间分布。北部以幕阜山脉为界，山地、丘陵占总面积的60%以上；中部为赣江冲积平原，占总面积的20%左右；南部为鄱阳湖湖区，占总面积的20%左右。

九江市主要山脉有幕阜山脉、九岭山脉、庐山等。主要河流有长江、赣江、修河、潦河等。主要湖泊有鄱阳湖、都昌湖、共山湖等。

气候

九江市属亚热带季风气候，四季分明，光热充足，雨量充沛。年平均气温16.9℃，年降水量1400毫米左右。

九江市夏季炎热多雨，冬季寒冷干燥。春季温暖多雨，秋季凉爽少雨。
